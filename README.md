## 📌 Project Overview

This project analyzes digital marketing campaign performance using an interactive Power BI dashboard. The dashboard helps evaluate campaign effectiveness, customer engagement, conversions, and return on investment (ROI).

---

## 🎯 Business Objective

The dashboard helps answer questions such as:

- Which campaigns generate the highest conversions?
- Which marketing channels perform best?
- How does customer engagement vary across campaigns?
- What are the factors influencing ROAS?
- What is the overall campaign ROI?
---

## 🛠️ Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Dashboard Features

- KPI Cards
- Campaign Performance Analysis
- Platform Performance
- Conversion Analysis
- Customer Segmentation
- Creative & Emotional Analysis
- Time Series Analysis
- Interactive Filters & Slicers
